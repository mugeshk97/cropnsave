# cropnsave

# Simple tool to crop and save the important areas in the document

`
python cropnsave.py
`

# keyBoard Shortcuts
<li>p - previous </li>
<li>n - next </li>
<li>Enter - crop </li>
<li>q - Quit </li>
<li>use mouse to mark coordinates press down (Left) to start and press up (Left) to end </li>
